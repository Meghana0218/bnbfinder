# 🏨 BnBFinder – Hotel Booking Web Application

BnBFinder is a hotel booking web application inspired by modern travel platforms like Airbnb.  
It allows users to search for hotels based on location and explore exciting travel destinations.

This project includes both a **frontend interface** and **backend functionality** for hotel search.

---

## 🚀 Features

- 🔍 Search hotels by **city or location**
- 🏨 View hotel listings
- 🌍 Explore popular destinations
- 📅 Select travel dates
- 👥 Choose number of guests
- 💻 Responsive and modern UI
- ⚡ Backend support for hotel search functionality

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

git clone https://github.com/Meghana0218/bnbfinder.git

### 2️⃣ Navigate to the project folder

cd bnbfinder

### 3️⃣ Install backend dependencies

npm install

### 4️⃣ Start the server

node server.js

---

## 🌐 Usage

1. Open the website in a browser  
2. Enter a **destination** in the search bar  
3. Click **Find Hotels**  
4. Browse available hotel listings
